# Changelog

## 1.0.0 — 2026-05-02

### Added

- `/speckit.decompose` command: parse a source document and generate a feature map with dependency ordering, priority classification, and foundation detection.
- `/speckit.decompose.select` command: interactively select the next feature to specify, with dependency validation and pre-drafted specify input.
- `/speckit.decompose.status` command: show decomposition progress with auto-update of feature map statuses based on existing spec artifacts.
- Feature map template with structured inventory table, dependency graph, and per-feature detail sections.
- SHA-256 source document tracking for drift detection.
- Support for MoSCoW tables, checkbox lists, numbered roadmaps, epic/story hierarchies, free-form prose PRDs, and technical spec documents.
